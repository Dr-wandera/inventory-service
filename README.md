Key Features

 Add stock for products for a  seller 

 Reduce stock when orders are placed

 Restore stock when orders are cancelled

 Check product availability before placing a product to cart

 Low-stock alerts 

  API Endpoints
POST   /api/v1/inventory/restock   → restore product stock
GET    /api/v1/inventory/checkProduct  → Get product by Id
GET    /api/v1/inventory/checkStock    → Get product stock
